# 🎬 Central Tickets Front

Frontend do projeto **Central Tickets**, desenvolvido com **Next.js (App Router)** e **Chakra UI v3**, focado em gerenciamento de abertura a chamados de suporte entre **Operação x Suporte**.

---

## 🚀 Tecnologias Principais

- **Next.js 16 (App Router)**
- **React 19**
- **Chakra UI v3**
- **Framer Motion**
- **React Hook Form + Yup**
- **Nookies**
- **Next Themes**

---

## 🛠️ Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```
